CEF 内核 GPU 加速测试入口

企业微信浏览器打开本页面，先看下方"GPU 自检"，再依次点击各测试用例。建议测试 GPU 开启 / 关闭 两种情况对比。

① GPU 自检

在 CEF 内嵌浏览器中也可以直接访问：chrome://gpu 查看完整 GPU 报告（如内核未禁用该页）。

① WebGL 复杂 3D 图形

WebGL2 + 1 万粒子 + 旋转立方体阵列，验证 3D 渲染、shader、纹理。

② Canvas2D 高负载绘制

5000 个移动小球 + 渐变 + 阴影 + 路径，验证 2D 加速。

③ 视频播放（多路 / 硬解）

H.264 / WebM 多路同播，4K 大码率，验证视频解码与合成。

④ 地图渲染

Leaflet + OSM（无需 Key），切换瓦片、缩放、平移压力。

⑤ CSS3D / 海量 DOM 动画

500 个 DOM 节点 transform3d 动画，验证合成层与 GPU 提升。

⑥ 综合压力（同页多场景）

WebGL + 视频 + Canvas 同时跑，长时间观察是否花屏 / 黑块 / 崩溃。
